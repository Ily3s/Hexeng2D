Detailed Description

An instance that can gather n quads on the same layer and render them together.

Note
A BatchInstance is a Mesh. If you set the pos, rotation or scale of the BatchInstance, it will add up with the pos, rotation or scale of each of the BatchQuads.
BatchInstance::texture must be a TextureAtlas*
