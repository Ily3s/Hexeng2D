Create streams to host this sound.

To play a sound, there must be a stream ready to host it. If there isn't, the sound is not played and 4 extra streams get created instead.

Note
By default (without using this function), 4 streams per sound are created.

Reimplemented in Hexeng::Audio::SpatialSound.
