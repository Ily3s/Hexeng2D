conditionA pointer to the "control variable" of the layer. If the bool it is pointing to is set to false, the layer will not show up, if it is set to true, the layer will show up.
z_posThe depth of the Layer.
posThe position mode of the layer (take a look at Position).
rangeIf set to Range::LOCAL, the layer needs to be added to a scene. If set to Range::GLOBAL, the layer will apear on all scenes.
