\doxysubsection{Detailed Description}
A \"{}copy\"{}, \"{}link\"{} to a \doxylink{classHexeng_1_1Renderer_1_1Mesh}{Mesh}. 

It can have different uniforms (position, color, texture ...) than the original mesh but the vertex array object stays the same. If the original mesh gets destructed, then all copies of it become unvalid (as they are more like links than like copies).
