\doxysubsection{Detailed Description}
Pass input directly to output. 

Note that some HW devices, for example many ISA audio cards on PCs, do NOT support full duplex! For a PC, you normally need a PCI based audio card such as the SBLive.

\begin{DoxyAuthor}{Author}
Phil Burk \href{http://www.softsynth.com}{\texttt{ http\+://www.\+softsynth.\+com}}
\end{DoxyAuthor}
While adapting to V19-\/\+API, I excluded configs with frames\+Per\+Callback=0 because of an assert in file pa\+\_\+common/pa\+\_\+process.\+c. Pieter, Oct 9, 2003.
