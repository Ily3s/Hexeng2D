\doxysubsection{Detailed Description}
\mbox{\hyperlink{classportaudio_1_1_system}{System}} singleton which represents the Port\+Audio system. 

The \mbox{\hyperlink{classportaudio_1_1_system}{System}} is used to initialize/terminate Port\+Audio and provide a single access point to the Port\+Audio \mbox{\hyperlink{classportaudio_1_1_system}{System}} (\mbox{\hyperlink{classportaudio_1_1_system_a460800fc5a870298eb4d3f281253ddf4}{instance()}}). It can be used to iterate through all \mbox{\hyperlink{classportaudio_1_1_host_api}{Host\+Api}} \textquotesingle{}s in the \mbox{\hyperlink{classportaudio_1_1_system}{System}} as well as all devices in the \mbox{\hyperlink{classportaudio_1_1_system}{System}}. It also provides some utility functionality of Port\+Audio.

Terminating the \mbox{\hyperlink{classportaudio_1_1_system}{System}} will also abort and close the open streams. The \mbox{\hyperlink{classportaudio_1_1_stream}{Stream}} objects will need to be deallocated by the client though (it\textquotesingle{}s usually a good idea to have them cleaned up automatically).
