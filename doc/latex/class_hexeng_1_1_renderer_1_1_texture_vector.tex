\doxysubsection{Detailed Description}
Allows to push multiple textures linearly to form a unique texture that is in fact a one dimentional array of textures.  

\begin{DoxyNote}{Note}
If the different textures are not the same size, this may be unoptimized for memory. Also if you\textquotesingle{}re looking for a texture for batch rendering or similar, you should use \mbox{\hyperlink{class_hexeng_1_1_renderer_1_1_texture_atlas}{Texture\+Atlas}} instead. 
\end{DoxyNote}
