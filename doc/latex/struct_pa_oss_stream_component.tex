\doxysubsection{Detailed Description}
Per-\/direction structure for \mbox{\hyperlink{struct_pa_oss_stream}{Pa\+Oss\+Stream}}.

Aspect Stream\+Channels\+: In case the user requests to open the same device for both capture and playback, but with different number of channels we will have to adapt between the number of user and host channels for at least one direction, since the configuration space is the same for both directions of an OSS device.
