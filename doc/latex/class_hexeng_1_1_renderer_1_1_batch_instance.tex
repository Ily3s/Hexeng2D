\doxysubsection{Detailed Description}
An instance that can gather 250 quads on the same layer at max and render them together.  

\begin{DoxyNote}{Note}
A \mbox{\hyperlink{class_hexeng_1_1_renderer_1_1_batch_instance}{Batch\+Instance}} is a \mbox{\hyperlink{class_hexeng_1_1_renderer_1_1_mesh}{Mesh}}. If you set the pos, rotation or scale of the \mbox{\hyperlink{class_hexeng_1_1_renderer_1_1_batch_instance}{Batch\+Instance}}, it will add up with the pos, rotation or scale of each of the Batch\+Quads. 
\end{DoxyNote}
