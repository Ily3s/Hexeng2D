\doxysubsection{Detailed Description}
Timing information for the buffers passed to the stream callback.

Time values are expressed in seconds and are synchronised with the time base used by \mbox{\hyperlink{portaudio_8h_a2b3fb60e6949f37f7f134105ff425749}{Pa\+\_\+\+Get\+Stream\+Time()}} for the associated stream.
