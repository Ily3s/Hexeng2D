\doxysubsection{Detailed Description}
To avoid name collision, everything in Port\+Audio\+Cpp is in the portaudio namespace. If this name is too long it\textquotesingle{}s usually pretty safe to use an alias like \`{}`namespace pa = portaudio;'\textquotesingle{}.
