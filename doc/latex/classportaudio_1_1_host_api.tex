\doxysubsection{Detailed Description}
\mbox{\hyperlink{classportaudio_1_1_host_api}{Host\+Api}} represents a host API (usually type of driver) in the \mbox{\hyperlink{classportaudio_1_1_system}{System}}. 

A single \mbox{\hyperlink{classportaudio_1_1_system}{System}} can support multiple \mbox{\hyperlink{classportaudio_1_1_host_api}{Host\+Api}}\textquotesingle{}s each one typically having a set of Devices using that \mbox{\hyperlink{classportaudio_1_1_host_api}{Host\+Api}} (usually driver type). All Devices in the \mbox{\hyperlink{classportaudio_1_1_host_api}{Host\+Api}} can be enumerated and the default input/output \mbox{\hyperlink{classportaudio_1_1_device}{Device}} for this \mbox{\hyperlink{classportaudio_1_1_host_api}{Host\+Api}} can be retrieved.
