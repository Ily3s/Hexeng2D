\doxysubsection{Detailed Description}
A pointer to a pa\+Mac\+Core\+Stream\+Info may be passed as the host\+Api\+Specific\+Stream\+Info in the \mbox{\hyperlink{struct_pa_stream_parameters}{Pa\+Stream\+Parameters}} struct when opening a stream or querying the format. Use NULL, for the defaults. Note that for duplex streams, flags for input and output should be the same or behaviour is undefined.
