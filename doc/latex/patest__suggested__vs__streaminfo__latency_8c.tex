\doxysubsection{Detailed Description}
Print suggested vs. \mbox{\hyperlink{struct_pa_stream_info}{Pa\+Stream\+Info}} reported actual latency. 

\begin{DoxyAuthor}{Author}
Ross Bencina \href{mailto:rossb@audiomulch.com}{\texttt{ rossb@audiomulch.\+com}}
\end{DoxyAuthor}
Opens streams with a sequence of suggested latency values from 0 to 2 seconds in .5ms intervals and gathers the resulting actual latency values. Output a csv file and graph suggested vs. actual. Run with frames\+Per\+Buffer unspecified, powers of 2 and multiples of 50 and prime number buffer sizes.
