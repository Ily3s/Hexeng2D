\doxysubsection{Detailed Description}
A \mbox{\hyperlink{classportaudio_1_1_stream}{Stream}} represents an active or inactive input and/or output data stream in the \mbox{\hyperlink{classportaudio_1_1_system}{System}}. 

Concrete \mbox{\hyperlink{classportaudio_1_1_stream}{Stream}} classes should ensure themselves being in a closed state at destruction (i.\+e. by calling their own \mbox{\hyperlink{classportaudio_1_1_stream_acb79f85f43b9f87b1e3c495d5f02ae58}{close()}} method in their deconstructor). Following good C++ programming practices, care must be taken to ensure no exceptions are thrown by the deconstructor of these classes. As a consequence, clients need to explicitly call \mbox{\hyperlink{classportaudio_1_1_stream_acb79f85f43b9f87b1e3c495d5f02ae58}{close()}} to ensure the stream closed successfully.

The \mbox{\hyperlink{classportaudio_1_1_stream}{Stream}} object can be used to manipulate the \mbox{\hyperlink{classportaudio_1_1_stream}{Stream}}\textquotesingle{}s state. Also, time-\/constant and time-\/varying information about the \mbox{\hyperlink{classportaudio_1_1_stream}{Stream}} can be retrieved.
