Create streams to host this sound. 

To play a sound, there must be a stream ready to host it. If there isn\textquotesingle{}t, the sound is not played and 4 extra streams get created instead. \begin{DoxyNote}{Note}
By default (without using this function), 4 streams per sound are created.
