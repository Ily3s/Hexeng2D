\doxysubsection{Detailed Description}
Represents a stream for blocking read/write I/O.

This Java object contains the pointer to a \mbox{\hyperlink{classcom_1_1portaudio_1_1_port_audio}{Port\+Audio}} stream stored as a long. It is passed to \mbox{\hyperlink{classcom_1_1portaudio_1_1_port_audio}{Port\+Audio}} when calling stream related functions.

To create one of these, call Port\+Audio.\+open\+Stream().
