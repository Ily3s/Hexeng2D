\doxysubsection{Detailed Description}
Java methods that call \mbox{\hyperlink{classcom_1_1portaudio_1_1_port_audio}{Port\+Audio}} via JNI. This is a portable audio I/O library that can be used as an alternative to Java\+Sound.

Please see the \mbox{\hyperlink{classcom_1_1portaudio_1_1_port_audio}{Port\+Audio}} documentation for a full explanation.

\href{http://portaudio.com/docs/}{\texttt{ http\+://portaudio.\+com/docs/}} \href{http://portaudio.com/docs/v19-doxydocs/portaudio_8h.html}{\texttt{ http\+://portaudio.\+com/docs/v19-\/doxydocs/portaudio\+\_\+8h.\+html}}

This Java binding does not support audio callbacks because an audio callback should never block. Calling into a Java virtual machine might block for garbage collection or synchronization. So only the blocking read/write mode is supported.
