\doxysubsection{Detailed Description}
An instance that can gather n quads on the same layer and render them together. 

\begin{DoxyNote}{Note}
A \doxylink{classHexeng_1_1Renderer_1_1BatchInstance}{Batch\+Instance} is a \doxylink{classHexeng_1_1Renderer_1_1Mesh}{Mesh}. If you set the pos, rotation or scale of the \doxylink{classHexeng_1_1Renderer_1_1BatchInstance}{Batch\+Instance}, it will add up with the pos, rotation or scale of each of the Batch\+Quads.
