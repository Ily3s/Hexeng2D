\doxysubsection{Detailed Description}
Definition of 16 and 32 bit integer types (Pa\+Int16, Pa\+Int32 etc) 

SIZEOF\+\_\+\+SHORT, SIZEOF\+\_\+\+INT and SIZEOF\+\_\+\+LONG are set by the configure script when it is used. Otherwise we default to the common 32 bit values, if your platform doesn\textquotesingle{}t use configure, and doesn\textquotesingle{}t use the default values below you will need to explicitly define these symbols in your make file.

A PA\+\_\+\+VALIDATE\+\_\+\+SIZES macro is provided to assert that the values set in this file are correct.
