\doxysubsection{Detailed Description}
The entire set of parameters needed to configure and open a \mbox{\hyperlink{classportaudio_1_1_stream}{Stream}}. 

It contains parameters of input, output and shared parameters. Using the \mbox{\hyperlink{classportaudio_1_1_stream_parameters_ad24fe8d02b719cf36a5ea55eaf982ec4}{is\+Supported()}} method, the \mbox{\hyperlink{classportaudio_1_1_stream_parameters}{Stream\+Parameters}} can be checked if opening a \mbox{\hyperlink{classportaudio_1_1_stream}{Stream}} using this \mbox{\hyperlink{classportaudio_1_1_stream_parameters}{Stream\+Parameters}} would succeed or not. Accessors are provided to higher-\/level parameters aswell as the lower-\/level parameters which are mainly intended for internal use.
